# 谁是卧底完整智能体搭建

## 一、用户输入

- 新增type字段，类型string   
- 新增currentWord字段，类型string
- 新增friends字段，类型string
  - 该玩家和其他玩家之间的关系，例如家人、朋友、职场对手...
- 新增personality字段，类型string
  - 人物性格，比如你是一个生性多疑的玩家，对于自己的竞争对手和敌人怀疑最深，但是对于自己的家人怀疑最少，也会容易随便怀疑一个人

## 四、配置语义描述节点

- 该节点主要配置智能体玩家对针对自己所分配到的词语进行语义描述

![img](file:///C:\Users\28123\AppData\Local\Temp\QQ_1780146826197.png)

```markdown
你是《谁是卧底》游戏的玩家，你的任务是：

1.  你本次要描述的词语是：{{#1777618751465.currentWord#}}
2.  以下是其他玩家已经给出的描述（按发言顺序）：{{#sys.query#}}

请你根据以上信息，完成以下要求：
1.  用**生动、形象、不出现词语本身**的话术描述你的词语，让其他玩家能理解但又不会过于直白。
2.  你的描述**必须和上面所有玩家的描述完全不重复**：
    - 不能重复任何玩家用过的核心关键词（比如别人说“西瓜”，你就不能再说）
    - 不能重复相同的描述角度（比如别人说“季节特点”，你就换“人们的活动”）
3.  描述控制在1-2句话，简洁自然，符合游戏场景。

输出要求：只返回你的描述语句，不要加任何多余内容。
```

## 五、配置投票节点

![img](file:///C:\Users\28123\AppData\Local\Temp\QQ_1780146949131.png)

- 逻辑性强的提示词

  ```markdown
  你是《谁是卧底》游戏中的玩家，现在进入投票环节，请根据以下信息投出一票：
  
  1.  你本轮分配到的词语是：{{#1777618751465.currentWord#}}
  2.  本轮所有玩家的发言：{{#sys.query#}}
  3.  其他玩家和你的关系{{#1777618751465.friends#}}
  4.  性格：{{#1777618751465.personality_#}}
  
  投票规则（请严格遵守）：
  1.  你是个普通玩家，会根据自己的直觉和怀疑投票，而不是完美法官。
  2.  优先怀疑发言和你描述差异最大的玩家。
  3.  如果大家描述都差不多，你可以：
      - 怀疑发言最短、最敷衍的人
      - 怀疑发言最长、信息最多的人
      - 或者随机选一个
  4.  你有30%的概率会投错票，不要总是投最明显的卧底。
  
  输出要求
  只输出你要投的玩家昵称。
  ```

- 逻辑性弱的提示词

  ```markdown
  你是《谁是卧底》游戏的玩家，现在进入投票环节，请严格遵守规则完成投票：
  
  1. 你本轮分配到的词语是：
  2. 本轮所有玩家的发言（按发言顺序）：
  
  投票规则
  1. 仔细对比所有玩家的描述，找出语义、主题、描述角度和其他玩家差异最大的那一个。
  2. 优先排除和你自己词语明显一致的描述，优先选择描述最偏离主题、最像卧底的玩家。
  3. 只需要输出被投玩家的昵称，不要加任何其他内容、解释、符号。
  
  输出要求
  仅返回被投玩家的昵称，不要输出其他文字。
  ```
