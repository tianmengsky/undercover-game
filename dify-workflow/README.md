# Dify 工作流配置

## 导入方式

1. 打开 Dify 控制台（http://localhost/app）
2. 点击「创建应用」→「导入应用」
3. 选择此目录下的 `.yml` 文件

## 工作流结构

单一应用 + 三路分支（通过 `inputs.type` 分流）：

| type | 功能 | 输出 |
|------|------|------|
| `词语生成` | 知识检索 → LLM 随机生成词对 | `{common_word, undercover_word}` |
| `语义描述` | LLM 根据人设 + 历史发言流式输出 | 逐 token 流式 + `<think>` 块 |
| `投票` | LLM 分析发言记录 → 选最可疑的玩家 | 玩家昵称 |

## 导出

在 Dify 控制台点击「导出」→ 下载 DSL 文件 → 覆盖此目录下的 `.yml` 文件。
